ACME Corp · Reporte RPT-2026-001 · 2026-01-11
Executive Security Report · Exposición pública · Confidencial
Hallazgos & Ruta de Cierre

Exposed Database Credentials CRITICAL

  • Impacto: Full database compromise possible.
  • Credentials for the production database were found in a public GitHub repository.
Acción: Rotate credentials immediately and audit access logs.

Admin Portal without MFA HIGH

  • Impacto: Unauthorized access potential.
  • The administrative portal at admin.acme.com does not enforce Multi-Factor Authentication.
Acción: Enforce MFA for all admin accounts.

Outdated SSL Certificate MEDIUM

  • Impacto: Man-in-the-middle attacks.
  • The SSL certificate for dev.acme.com expires in 2 days.
Acción: Renew certificate.
